# NCCUCourseRate
A simple python base web crawler for nccu course rate

## Environment
- Python 3.9^

## Installation
```sh
pip3 install -r requirement.txt
```

## Setup
As a NCCU student, you have your iNCCU account. Please create a `.env` file and give following informations.
```
USERNAME=*********
PASSWORD=*****
```

## Execution
```sh
python3.9 main.py
```
